# トクミル プライバシーポリシー

最終更新日: 【YYYY年M月D日】
制定日: 【YYYY年M月D日】

Ringed Seal Studio（以下「当方」といいます）は、モバイルアプリケーション「トクミル」およびこれに付随するサービス（以下「本サービス」といいます）における利用者（以下「ユーザー」といいます）の情報の取扱いについて、本プライバシーポリシー（以下「本ポリシー」といいます）を定めます。

---

## 1. 本サービスにおける情報の特性

本サービスは匿名認証（Firebase Anonymous Authentication）を採用しており、**氏名・住所・電話番号・生年月日等の直接的な個人識別情報を収集しません**。当方が取得する情報は、匿名ユーザーID（UID）、ユーザーが本サービス内で記録したスキャン履歴・購入履歴、プッシュ通知用の端末識別子（FCMトークン）等に限られます。これにより、万一情報の漏えいが発生した場合でも、特定の個人を識別するリスクは構造的に最小限となるよう設計しています。

## 2. 取得する情報・利用目的・提供先

| 取得する情報 | 利用目的 | 第三者提供先 | 保存場所 |
|---|---|---|---|
| 匿名ユーザーID（UID） | ユーザーの識別、データの紐付け | Google（Firebase） | Firebase（東京リージョン） |
| カメラ映像・撮影画像 | 商品・バーコード・レシートの認識（解析のための一時送信のみ。当方サーバーには保存しない） | Google（Gemini API / 画像認識） | 端末内および解析時の一時処理のみ |
| スキャン履歴・閲覧履歴・マイリスト | 機能の提供、過去履歴の表示 | なし | Firebase Firestore またはユーザー端末内 |
| 購入履歴・節約額・レシート情報（店舗名・商品名・金額） | 節約レポート機能の提供、価格比較 | なし | Firebase Firestore またはユーザー端末内 |
| FCMトークン（端末識別子） | プッシュ通知の配信 | Google（Firebase） | Firebase Firestore |
| 位置情報（許可した場合のみ） | 近隣店舗の価格比較・スキャン体験の補助 | なし | 端末内（取得時のみ利用） |
| サブスクリプション購入情報 | 課金状態の管理、レシート検証 | RevenueCat, Inc.（米国） | RevenueCat サーバー |

本サービスは、利用目的の達成に必要な範囲を超えて情報を取得・利用しません（個人情報の保護に関する法律第18条）。

## 3. 第三者提供および外国にある第三者への移転

1. 当方は、法令に基づく場合を除き、ユーザーの情報を本人の同意なく第三者に提供しません。
2. 本サービスは、機能の提供のために以下の外部サービスを利用しており、これに伴いユーザーの情報が外国にある第三者へ移転される場合があります。
   - **Google LLC（米国・Firebase / Gemini API）**: 認証基盤、データベース、画像認識、プッシュ通知
   - **RevenueCat, Inc.（米国）**: サブスクリプションの課金管理・レシート検証
3. これらの事業者は各自のプライバシーポリシーに基づき情報を取り扱います。米国の個人情報保護制度に関する情報は、個人情報保護委員会の公表資料等をご参照ください。

## 4. 安全管理措置

当方は、取得した情報の漏えい・滅失・毀損を防止するため、以下の安全管理措置を講じます。

- Firebase のセキュリティルールによるアクセス制御（原則として本人のデータのみ読み書き可能）
- HTTPS による通信の暗号化
- APIキー等の秘匿情報をクラウドのシークレット管理機構で管理し、アプリバイナリへ埋め込まない
- 認証済みリクエストのみを許可するサーバー構成

## 5. 漏えい等が発生した場合の対応

個人データの漏えい・滅失・毀損等（以下「漏えい等」といいます）が発生した場合、当方は次の対応を行います。

1. 影響範囲の特定と被害拡大防止措置
2. 個人情報保護委員会への報告（個人情報保護法第26条に基づき、要報告事案に該当する場合）
3. 影響を受けるユーザーへの周知（本ポリシーページ、各ストアのリリースノート、アプリ内通知等による）

なお、本サービスは匿名認証を採用しているため、特定の個人への直接の連絡が困難な場合があります。その場合は本ページおよび各ストアでの公表をもって通知に代えることがあります。

## 6. 保存期間

| 情報 | 保存期間 |
|---|---|
| スキャン履歴・閲覧履歴・マイリスト | アカウント削除またはアンインストールまで |
| 購入履歴・節約額・レシート情報 | アカウント削除まで |
| FCMトークン | アカウント削除まで（または長期間未使用の場合は当方の判断で削除） |
| サブスクリプション購入情報 | 課金管理に必要な期間 |

## 7. ユーザーの権利

ユーザーは、自己の保有個人データについて、開示・訂正・追加・削除・利用停止・消去・第三者提供の停止を請求できます（個人情報保護法第32条〜第39条）。請求は第10項の問い合わせ窓口までご連絡ください。また、本サービスの設定画面の「アカウントを削除する」機能により、当方が取得したデータを削除できます。

## 8. 未成年者の利用

本サービスは13歳以上の方を対象としています。13歳未満の方は利用できません。保護者の方は、お子様の利用状況をご確認ください。

## 9. アフィリエイトの開示

本サービスは、Amazonアソシエイト・プログラム、楽天アフィリエイト、バリューコマース（Yahoo!ショッピング）等のアフィリエイトプログラムに参加しています。本サービス内の商品リンクを経由して購入が行われた場合、当方は各プログラムから一定の紹介報酬を受け取ることがあります。当該報酬は本サービスの運営・改善に利用されます。

当方は、法令の改正やサービス内容の変更等に応じて本ポリシーを変更することがあります。重要な変更を行う場合は、本ページでの掲示、アプリ内通知、各ストアのリリースノート等によりお知らせします。変更後に本サービスを継続して利用された場合、変更後の本ポリシーに同意したものとみなします。
