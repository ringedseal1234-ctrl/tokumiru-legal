制定日: 2026年6月19日

本利用規約（以下「本規約」といいます）は、Ringed Seal Studio（以下「当方」といいます）が提供するモバイルアプリケーション「トクミル」およびこれに付随するサービス（以下「本サービス」といいます）の利用条件を定めるものです。

## 第1条（適用）

1. 本規約は、本サービスの利用に関する当方とユーザーとの間の一切の関係に適用されます。
2. 本サービス内で個別の注意事項やガイドラインを表示した場合、それらは本規約の一部を構成します。
3. ユーザーは、本規約および[プライバシーポリシー](privacy.html)へ同意したうえで本サービスを利用するものとします。

## 第2条（本サービス）

1. 本サービスは、商品バーコード、商品画像、商品名などをもとに、ECサイトや店頭の価格情報を比較し、マイリスト、購入候補、節約記録、近隣価格などを管理・表示する機能を提供します。
2. 商品認識、画像認識、文字認識、商品名寄せ、価格比較には、機械学習、外部API、ユーザー入力等を使用します。
3. 本サービスが表示する価格、送料、在庫、内容量、商品画像、店舗、位置その他の情報は参考情報です。実際の取引条件は販売元で確認してください。
4. 本サービスの一部機能は、対応地域、端末、OS、外部サービス、データの蓄積状況によって利用できない場合があります。
5. 当方は、本サービスの内容を追加、変更、制限または終了することがあります。

## 第3条（利用資格）

1. 本サービスは13歳以上の方を対象とします。
2. 未成年者は、必要に応じて親権者その他の法定代理人の同意を得て利用するものとします。
3. 本規約へ同意できない場合、本サービスを利用できません。

## 第4条（アカウント）

1. 本サービスは初回利用時にFirebaseの匿名認証を使用します。
2. ユーザーは、任意でメールアドレスとパスワードを登録し、匿名アカウントを保存できます。
3. ユーザーは、登録情報を正確に管理し、パスワードを第三者へ開示しないものとします。
4. メールアカウントへ保存していない状態でアプリを削除、端末を初期化または変更した場合、データを復元できないことがあります。
5. 不正利用、規約違反、セキュリティ上の問題がある場合、当方はアカウントまたは機能を制限することがあります。

## 第5条（料金）

1. 現在の提供機能には無料で利用できる範囲があります。当方は、将来、有料機能またはサブスクリプションを提供する場合があります。
2. 有料機能を提供する場合、料金、期間、自動更新、解約方法その他の条件を購入画面へ表示します。
3. アプリ内課金はApple App StoreまたはGoogle Playを通じて行い、決済・返金には各ストアの規約が適用されます。
4. 通信料金、端末費用その他本サービスを利用するための費用はユーザーが負担します。

## 第6条（価格・商品・店舗情報）

1. 外部ECサイトや店舗の価格・在庫・送料・ポイント・商品情報は、取得後に変更される場合があります。
2. 同じ商品名でも、容量、数量、型番、色、版、販売条件等が異なる場合があります。
3. AIやOCRによる商品・価格の認識結果には誤りが含まれる場合があります。
4. 地域価格は、ユーザー入力、画像認識、外部データ等をもとに登録・集計されます。当方は、すべての情報の正確性、完全性、最新性を保証しません。
5. 購入前に、販売元の商品ページ、棚札、レジ表示等で最終条件を確認するものとします。

## 第7条（ユーザーによる情報の登録）

1. ユーザーは、店頭価格、店舗、商品その他の情報を登録する場合、確認できた事実に基づいて入力するものとします。
2. ユーザーは、第三者の権利を侵害する情報、虚偽情報、個人情報、機密情報その他不適切な情報を登録してはなりません。
3. 当方は、品質、安全性、法令遵守、運用上の必要性のため、登録情報を非表示、修正、集計対象外または削除することがあります。
4. ユーザーは、本サービスの提供・改善・集計に必要な範囲で、登録情報を当方が利用することを許諾します。

1. 家族共有機能では、招待されたメンバー間で購入記録や節約情報等が共有される場合があります。
2. 招待コードやアカウント情報は、信頼できる相手とのみ共有してください。
3. グループ内で共有された情報は、他のメンバーから閲覧されます。
4. グループへの参加・退出、端末変更、アカウント削除等により、共有情報へアクセスできなくなる場合があります。

## 第9条（アフィリエイト）

1. 本サービス内の一部の外部リンクには、アフィリエイト情報が含まれる場合があります。
2. リンクを経由して購入された場合、当方が楽天アフィリエイト、バリューコマース、Amazonアソシエイト等から紹介料を受け取る場合があります。

4. 商品、配送、返品、返金、決済等の問い合わせは販売事業者へ行うものとします。

2. 当方または第三者の権利・利益を侵害する行為

8. 本サービスの情報を許可なく転載、再配布または商業利用する行為
