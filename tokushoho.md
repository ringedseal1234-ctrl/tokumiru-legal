> 現在、トクミルの有料プランは提供準備中です。有料機能の提供開始前に、実際の販売条件・料金・事業者情報へ更新します。

| 項目 | 内容 |
|---|---|
| 販売事業者 | Ringed Seal Studio |
| 運営責任者 | 請求があった場合、遅滞なく開示します |
| 所在地 | 請求があった場合、遅滞なく開示します |
| 電話番号 | 請求があった場合、遅滞なく開示します。問い合わせはメールで受け付けます |
| メールアドレス | ringedseal1234+support@gmail.com |
| 販売価格 | 有料機能の提供開始時に、アプリ内の購入画面へ消費税込みの価格を表示します |
| 商品代金以外の必要料金 | インターネット接続に必要な通信料金、端末費用等はユーザーの負担です |
| 支払方法 | Apple App StoreまたはGoogle Playが提供する支払方法 |
| 支払時期 | 各ストアおよび購入画面に表示される条件に従います |
| 役務の提供時期 | 決済完了後、利用可能になります |
| 自動更新 | サブスクリプションを提供する場合、期間終了前に解約しない限り自動更新されます。更新条件は購入画面へ表示します |
| 解約 | 各ストアのアカウント設定から解約できます。アプリの削除だけでは解約されません |
| 返金・キャンセル | デジタルサービスの性質上、提供開始後の返金は原則として行いません。ただし、法令または各ストアの返金ポリシーに定めがある場合を除きます |
| 動作環境 | 購入画面および本ウェブサイトで案内します |
